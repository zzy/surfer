# Web Application Server - yew

``` Bash
git clone https://github.com/zzy/surfer.git
cd surfer
cargo build

cd frontend-yew
```

## Build & Run:

``` Bash
cargo install trunk wasm-bindgen-cli

trunk build
trunk serve --release
```
Then connect to http://127.0.0.1:3001 with browser.
